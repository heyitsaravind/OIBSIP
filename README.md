# OIBSIP - Oasis Infobyte Internship Projects

This repository contains projects completed during my Java Developer internship at Oasis Infobyte.

## Task 1: ATM Interface System
**Location:** `Task1-Enhanced-ATM/`

Console-based ATM system with banking operations.

**Features:**
- Login with PIN
- Transaction history
- Withdraw, Deposit, Transfer
- Logout

**Run:**
```bash
cd ~/Documents/Java\ Apps/DIGITAL\ LIBRARY\ MANAGEMENT
cd OIBSIP_JavaDevelopement_Task1
javac *.java
java ATMInterface
```
