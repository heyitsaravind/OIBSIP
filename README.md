### Task 1: ATM Interface
**Location:** `Task1-ATM-Interface/`

A comprehensive console-based ATM system built in Java demonstrating:
- User authentication with PIN validation
- Core banking operations (withdraw, deposit, transfer)
- Transaction history tracking
- Object-oriented programming principles
- Error handling and input validation

**Features:**
- Secure login system
- Real-time balance management
- Complete transaction audit trail
- Multiple test accounts
- Professional console interface

**How to Run:**
```bash
cd Task1-ATM-Interface
javac *.java
java ATMSystem
```
